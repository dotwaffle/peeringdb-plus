<objective>
Add `entrest.WithDefaultSort("updated")` + `entrest.WithDefaultOrder(entrest.OrderDesc)` annotations to all 13 ent schema `Annotations()` methods, and add `entrest.WithSortable(true)` to each entity's `updated` field. Update the schema-generator template at `cmd/pdb-schema-generate/main.go` in lock-step so regeneration is idempotent.

Purpose: These annotations declare the single-field entrest default sort for REST (`/rest/v1/*`) list endpoints. By themselves they give `updated DESC` but NOT the compound `(-updated, -created, -id)` — the compound default is delivered by Plan 03 (TemplateDir override). This plan is the declarative foundation plus documentation for the `?sort=` override contract.

Output: 13 schemas with new annotations; schema-generator template updated; `go generate ./...` regenerates `ent/rest/sorting.go` to show `DefaultField: "updated"` / `DefaultOrder: "desc"` on all 13 `*SortConfig` blocks.
</objective>
