**4. In doWithRetry() -- add per-attempt spans per D-03, rate limiter events per D-05:**

CRITICAL (Pitfall 3): Use a DIFFERENT variable name `attemptCtx` for the attempt span context to avoid shadowing `ctx`. Each attempt span must be a sibling (child of FetchAll), not chained.

Replace the loop body. The new structure:
```go
for attempt := range maxRetries {
    // Honor context cancellation between retries.
    if err := ctx.Err(); err != nil {
        return nil, fmt.Errorf("fetch %s: %w", url, err)
    }

    attemptCtx, attemptSpan := tracer.Start(ctx, "peeringdb.request",
        trace.WithAttributes(
            attribute.Int("http.request.resend_count", attempt),
        ),
    )

    // Wait for rate limiter.
    waitStart := time.Now()
    if err := c.limiter.Wait(attemptCtx); err != nil {
        attemptSpan.End()
        return nil, fmt.Errorf("rate limiter for %s: %w", url, err)
    }
    waitDuration := time.Since(waitStart)
    if waitDuration > time.Millisecond {
        attemptSpan.AddEvent("rate_limiter.wait",
            trace.WithAttributes(
                attribute.Float64("wait_duration_ms", float64(waitDuration.Milliseconds())),
            ),
        )
    }

    req, err := http.NewRequestWithContext(attemptCtx, http.MethodGet, url, nil)
    // ... rest of existing logic, but use attemptCtx for the request ...
    // End attemptSpan at each return/continue point
```

Key changes inside the loop:
- Create `attemptCtx, attemptSpan` using `tracer.Start(ctx, ...)` (NOT `attemptCtx` -- always start from the outer `ctx` so attempts are siblings)
- Pass `attemptCtx` to `http.NewRequestWithContext` so otelhttp transport spans nest under the attempt span
- Call `attemptSpan.End()` at every exit point (success return, error return, retry continue)
- On success: `attemptSpan.End()` then `return resp, nil`
- On retryable error: `attemptSpan.RecordError(lastErr)`, `attemptSpan.End()`, then continue
- On non-retryable error: `attemptSpan.RecordError(err)`, `attemptSpan.End()`, then return
- After loop exhausts retries: no span to end (already ended in loop)

Add `tracer := otel.Tracer("peeringdb")` at the top of doWithRetry.
