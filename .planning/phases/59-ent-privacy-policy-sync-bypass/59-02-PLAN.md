<objective>
Add `PDBPLUS_PUBLIC_TIER` env var parsing to `internal/config/config.go`. This is the second foundation brick: the HTTP middleware in Plan 03 reads the resolved `cfg.PublicTier` at construction and stamps every inbound request with it.

Per D-11/D-12, parsing is strict and case-sensitive lowercase: empty → `TierPublic`, `"public"` → `TierPublic`, `"users"` → `TierUsers`, anything else → fail-fast at startup. Mirrors the existing `parseSyncMode` pattern verbatim.

Purpose: GO-CFG-1 (fail-fast config) — a misconfigured process must not start and silently serve wrong-tier data.

Output: One `Config` field, one parser helper, four tests.

       ```go
       // parsePublicTier reads key from the environment and maps its value to a

       // and "users" are accepted (D-12). Anything else is a hard error so the
       // process fails fast at startup per GO-CFG-1.
       func parsePublicTier(key string, defaultVal privctx.Tier) (privctx.Tier, error) {
           v := os.Getenv(key)
           switch v {
           case "":
               return defaultVal, nil
           case "public":
               return privctx.TierPublic, nil
           case "users":
               return privctx.TierUsers, nil
           default:
               return 0, fmt.Errorf("invalid value %q for %s: must be 'public' or 'users'", v, key)
           }
       }
       ```
