    Add a `parseFloat64` helper function (similar to existing parseBool/parseDuration):
    ```go
    func parseFloat64(key string, defaultVal float64) (float64, error) {
        v := os.Getenv(key)
        if v == "" {
            return defaultVal, nil
        }
        f, err := strconv.ParseFloat(v, 64)
        if err != nil {
            return 0, fmt.Errorf("invalid float %q for %s: %w", v, key, err)
        }
        return f, nil
    }
    ```

    `Setup(ctx context.Context, in SetupInput) (*SetupOutput, error)` function:
    1. Call `buildResource(ctx, in.ServiceName)` to create the OTel resource
    2. Create TracerProvider:
       - `autoexport.NewSpanExporter(ctx)` for the exporter
       - `sdktrace.NewTracerProvider` with `WithBatcher(spanExporter)`, `WithResource(res)`, `WithSampler(sdktrace.TraceIDRatioBased(in.SampleRate))` per D-02
       - `otel.SetTracerProvider(tp)`
    3. Create MeterProvider:
       - `autoexport.NewMetricReader(ctx)` for the reader
       - `metric.NewMeterProvider` with `WithResource(res)`, `WithReader(metricReader)`
       - `otel.SetMeterProvider(mp)`
    4. Create LoggerProvider:
       - `autoexport.NewLogExporter(ctx)` for the exporter
       - `log.NewLoggerProvider` with `WithResource(res)`, `WithProcessor(log.NewBatchProcessor(logExporter))`
    5. Set W3C TraceContext propagation per D-09:
       - `otel.SetTextMapPropagator(propagation.NewCompositeTextMapPropagator(propagation.TraceContext{}, propagation.Baggage{}))`
    6. Start Go runtime metrics per D-05:
       - `runtime.Start(runtime.WithMeterProvider(mp))`
    7. Return `&SetupOutput{LogProvider: lp, Shutdown: compositeShutdown}` where compositeShutdown calls `lp.Shutdown`, `mp.Shutdown`, `tp.Shutdown` in reverse order, joining errors with `errors.Join`.

    `buildResource(ctx context.Context, serviceName string) *resource.Resource` function per D-08, D-10:
    1. Get version from `debug.ReadBuildInfo()`. If `info.Main.Version == "(devel)"`, try `vcs.revision` setting and take first 7 chars.
    2. Build attrs: `semconv.ServiceName(serviceName)`, `semconv.ServiceVersion(version)`
    3. Check env vars `FLY_REGION`, `FLY_MACHINE_ID`, `FLY_APP_NAME` and add `attribute.String("fly.region", ...)` etc. for each that is non-empty per D-10.
    4. `resource.Merge(resource.Default(), resource.NewWithAttributes(semconv.SchemaURL, attrs...))` and return.

    **Dual slog logger (internal/otel/logger.go) per D-03:**

    Imports: `"context"`, `"io"`, `"log/slog"`, `"go.opentelemetry.io/contrib/bridges/otelslog"`, `sdklog "go.opentelemetry.io/otel/sdk/log"`

    `NewDualLogger(stdout io.Writer, logProvider *sdklog.LoggerProvider) *slog.Logger`:
    - Create `otelHandler` via `otelslog.NewHandler("peeringdb-plus", otelslog.WithLoggerProvider(logProvider))`
    - Create `stdoutHandler` via `slog.NewJSONHandler(stdout, &slog.HandlerOptions{Level: slog.LevelInfo})`
    - Return `slog.New(&fanoutHandler{handlers: []slog.Handler{stdoutHandler, otelHandler}})`

    `fanoutHandler` struct with `handlers []slog.Handler`:
    - `Enabled(ctx context.Context, level slog.Level) bool` -- returns true if ANY handler is enabled
    - `Handle(ctx context.Context, record slog.Record) error` -- calls Handle on each enabled handler, best-effort (continues on individual handler error)
    - `WithAttrs(attrs []slog.Attr) slog.Handler` -- returns new fanoutHandler with WithAttrs applied to each sub-handler
    - `WithGroup(name string) slog.Handler` -- returns new fanoutHandler with WithGroup applied to each sub-handler
