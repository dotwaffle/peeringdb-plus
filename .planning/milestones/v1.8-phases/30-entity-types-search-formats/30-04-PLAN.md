Implement WHOIS-style output format (?format=whois) using RPSL-like key-value format for all entity types, and verify JSON mode returns complete data including child entity rows.

Purpose: WHOIS format is designed for network automation scripts that parse whois-like output. This is a key differentiator for the terminal interface, enabling PeeringDB data to be consumed by the same tools that parse RIPE/ARIN whois responses. JSON mode completeness ensures scripts using ?format=json get the same child entity data as the terminal view.

Output: Working `curl /ui/asn/13335?format=whois` producing RPSL-like output, and `curl /ui/asn/13335?format=json` including IX/facility rows.

**Create `whois.go`** with WHOIS format helpers and per-entity renderers (D-10 through D-15):

```go
// whoisKeyWidth is the column width for RPSL-style key alignment.
// Standard whois output uses keys followed by colon then spaces to column 16.
const whoisKeyWidth = 16

// writeWHOISField writes a single RPSL-formatted key-value line.
// Empty values are omitted. Key is left-aligned, padded to whoisKeyWidth.
func writeWHOISField(buf *strings.Builder, key, value string)

// writeWHOISMulti writes multiple values for the same key (RPSL repeated key convention per D-14).
func writeWHOISMulti(buf *strings.Builder, key string, values []string)

// writeWHOISHeader writes the standard WHOIS header comments (D-15).
func writeWHOISHeader(buf *strings.Builder, query string)
```

`writeWHOISField` implementation:
```go
func writeWHOISField(buf *strings.Builder, key, value string) {
    if value == "" {
        return
    }
    buf.WriteString(fmt.Sprintf("%-*s%s\n", whoisKeyWidth, key+":", value))
}
```

`writeWHOISMulti` implementation:
```go
func writeWHOISMulti(buf *strings.Builder, key string, values []string) {
    for _, v := range values {
        writeWHOISField(buf, key, v)
    }
}
```

`writeWHOISHeader` implementation:
```go
func writeWHOISHeader(buf *strings.Builder, query string) {
    buf.WriteString("% Source: PeeringDB-Plus\n")
    buf.WriteString(fmt.Sprintf("%% Query: %s\n", query))
    buf.WriteString("\n")
}
```

```go
switch d := data.(type) {

}
```
