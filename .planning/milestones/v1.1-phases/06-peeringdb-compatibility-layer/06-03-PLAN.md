    ```go
    func getOrgWithDepth(ctx context.Context, client *ent.Client, id, depth int) (any, error) {
        q := client.Organization.Query().Where(organization.ID(id))
        if depth >= 2 {
            q = q.WithNetworks().WithFacilities().WithInternetExchanges().WithCarriers().WithCampuses()
        }
        org, err := q.Only(ctx)
        if err != nil { return nil, err }

        base := organizationFromEnt(org)
        if depth < 2 {
            return base, nil
        }

        m := toMap(base)
        m["net_set"] = orEmptySlice(networksFromEnt(org.Edges.Networks))
        m["fac_set"] = orEmptySlice(facilitiesFromEnt(org.Edges.Facilities))
        m["ix_set"] = orEmptySlice(internetExchangesFromEnt(org.Edges.InternetExchanges))
        m["carrier_set"] = orEmptySlice(carriersFromEnt(org.Edges.Carriers))
        m["campus_set"] = orEmptySlice(campusesFromEnt(org.Edges.Campuses))
        return m, nil
    }
    ```

    Helper `orEmptySlice` ensures nil slices become `[]any{}` (empty JSON arrays, never null).

    **search.go** (per D-13, D-14, D-23):

    **Search (?q=) implementation:**
    `func buildSearchPredicates(search string, searchFields []string) []func(*sql.Selector)`
    For each field in searchFields, create a `sql.FieldContainsFold(field, search)` predicate. Combine all with OR logic. Return a single predicate that ORs all field matches:
    ```go
    func buildSearchPredicates(search string, searchFields []string) func(*sql.Selector) {
        return func(s *sql.Selector) {
            var ors []*sql.Predicate
            for _, f := range searchFields {
                ors = append(ors, sql.ContainsFold(f, search))
            }
            s.Where(sql.Or(ors...))
        }
    }
    ```

    If the search string is empty, return nil (no predicate applied).

    **Field projection (?fields=) implementation** (per D-14):
    `func applyFieldProjection(data []any, fields []string) []any`
    For each object in the data slice:
    1. Marshal to map[string]any
    2. Create new map with only the requested fields
    3. Always include "id" even if not explicitly requested
    Return the filtered slice.

    If fields is empty, return data unchanged.

    Per the research recommendation: ?fields= applies only to top-level objects, not to nested _set objects at depth > 0.
