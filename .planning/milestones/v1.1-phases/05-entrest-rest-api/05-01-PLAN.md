**Part A: Add entrest schema annotations to all 13 schemas' Annotations() methods (per D-05)**

Add import `"github.com/lrstanley/entrest"` to each schema file.

Add ONE entrest annotation to each schema's `Annotations()` method, keeping existing entgql annotations:

```go
func (TypeName) Annotations() []schema.Annotation {
    return []schema.Annotation{
        entgql.RelayConnection(),  // existing
        entgql.QueryField(),       // existing
        entrest.WithIncludeOperations(entrest.OperationRead, entrest.OperationList), // NEW per D-04, D-05
    }
}
```
