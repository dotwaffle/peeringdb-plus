- Create `testutil.SetupClient(t)` for isolated SQLite

- Instantiate `&XxxService{Client: client}` directly
- Table-driven tests with `wantLen int` and `wantErr connect.Code`
- Assert `len(resp.GetEntities())` matches `wantLen`
- Assert at least one field value on the first result for non-error cases

Stream test pattern (HTTP/2 test server):
- Create `setupXxxStreamServer(t, client)` helper returning `peeringdbv1connect.XxxServiceClient`
- Helper creates `&XxxService{Client: client, StreamTimeout: 30 * time.Second}`
- Uses `httptest.NewUnstartedServer`, `srv.EnableHTTP2 = true`, `srv.StartTLS()`
- Table-driven tests with `wantLen int`
- Drain stream via `for stream.Receive()`, count messages
- Assert `stream.Msg()` is not nil, assert at least one field value on first message
- Check `stream.Err()` after drain
