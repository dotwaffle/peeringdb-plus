    ```go

    type MapMarker struct {
        // Lat is the marker's latitude.
        Lat float64
        // Lng is the marker's longitude.
        Lng float64
        // Name is the display name in the popup.
        Name string
        // Location is the city/country text in the popup.
        Location string
        // NetCount is the network count shown in the popup.
        NetCount int
        // IXCount is the IX count shown in the popup.
        IXCount int
        // DetailURL is the link to the entity detail page.

    }
    ```

    Define the `initMap` templ script function (per D-04, D-06, D-07, D-09, D-10, D-12):
    ```
    script initMap(elementID string, lat float64, lng float64, zoom int, popupHTML string)
    ```
    The script body must:
    - Read current dark mode: `var isDark = document.documentElement.classList.contains('dark');`
    - Set tile URLs: light = `https://{s}.basemaps.cartocdn.com/rastertiles/voyager/{z}/{x}/{y}{r}.png`, dark = `https://{s}.basemaps.cartocdn.com/dark_all/{z}/{x}/{y}{r}.png`
    - Create map: `L.map(elementID, { scrollWheelZoom: false }).setView([lat, lng], zoom)`
    - Create tile layer with attribution `'&copy; <a href="https://www.openstreetmap.org/copyright">OpenStreetMap</a> contributors &copy; <a href="https://carto.com/attributions">CARTO</a>'`, subdomains `'abcd'`, maxZoom 19
    - Add marker: `L.marker([lat, lng]).addTo(map).bindPopup(popupHTML)` (popup closed by default per D-12)
    - Enable scroll zoom on click: `map.on('click', function() { map.scrollWheelZoom.enable(); })`
    - Disable scroll zoom on mouseout: `map.on('mouseout', function() { map.scrollWheelZoom.disable(); })`
    - Register for dark mode swap: `window.__pdbMaps = window.__pdbMaps || []; window.__pdbMaps.push({ tileLayer: tileLayer, lightURL: lightURL, darkURL: darkURL });`

    Define the `MapContainer` templ component:
    ```
    templ MapContainer(id string, markers []MapMarker, zoom int)
    ```
    - Renders a container div: `<div id={id} class="w-full h-[200px] md:h-[350px] rounded-lg border border-neutral-200 dark:border-neutral-700 overflow-hidden" role="application" aria-label="Map showing facility location"></div>`
    - Per D-02: 200px mobile, 350px desktop
    - Per D-03: full width, rounded corners
    - Per D-21: role="application", aria-label
    - For Phase 44 (single marker), calls `@initMap(id, markers[0].Lat, markers[0].Lng, zoom, buildPopupHTML(markers[0]))` after the div
    - Guard: only call initMap if `len(markers) > 0`

    Define `buildPopupHTML(m MapMarker) string` helper (pure Go function):
    - Uses inline styles for Leaflet popup DOM (per UI-SPEC popup contract)
    - Structure: facility name (semibold 14px), location (12px #666), counts (12px #666), "View facility ->" link (12px #10b981)
    - Uses `html.EscapeString` from `html` stdlib on name/location to prevent XSS
    - Format:
      ```
      <div>
        <div style="font-weight:600;font-size:14px;margin-bottom:4px;">{name}</div>
        <div style="font-size:12px;color:#666;margin-bottom:4px;">{location}</div>
        <div style="font-size:12px;color:#666;margin-bottom:8px;">{netCount} Networks &middot; {ixCount} IXPs</div>
        <a href="{detailURL}" style="font-size:12px;color:#10b981;">View facility &rarr;</a>
      </div>
      ```
    - Omit location line if Location is empty
    - Omit counts line if both NetCount and IXCount are 0
