Dashboard JSON skeleton (from research -- use as starting point):
- `__inputs` array with DS_PROMETHEUS input
- `__requires` array with grafana, prometheus, panel types
- `id: null`, `uid: "pdbplus-overview"`, `version: null`, `schemaVersion: 39`
- `templating.list`: $datasource (type: datasource), $type (label_values query), $region (label_values query)
- `time: { "from": "now-6h", "to": "now" }`, `refresh: "30s"`

Row 1 (Sync Health) PromQL:
- Freshness: `pdbplus_sync_freshness_seconds`
- Duration p95: `histogram_quantile(0.95, rate(pdbplus_sync_duration_seconds_bucket[$__rate_interval]))`
- Success rate: `sum(rate(pdbplus_sync_operations_total{status="success"}[$__rate_interval])) / sum(rate(pdbplus_sync_operations_total[$__rate_interval]))`
- Operations over time: `sum by(status)(rate(pdbplus_sync_operations_total[$__rate_interval]))`
- Fallback events: `sum(increase(pdbplus_sync_type_fallback_total[$__rate_interval]))`

Row 2 (HTTP RED) PromQL:
- Request rate: `sum by(http_route)(rate(http_server_request_duration_seconds_count[$__rate_interval]))`
- Error rate: `sum(rate(http_server_request_duration_seconds_count{http_response_status_code=~"5.."}[$__rate_interval])) / sum(rate(http_server_request_duration_seconds_count[$__rate_interval]))`
- Latency p95: `histogram_quantile(0.95, sum by(le)(rate(http_server_request_duration_seconds_bucket[$__rate_interval])))`
- Active requests: `http_server_active_requests`

Row 3 (Per-Type Sync) PromQL:
- Duration by type: `histogram_quantile(0.95, sum by(type, le)(rate(pdbplus_sync_type_duration_seconds_bucket{type=~"$type"}[$__rate_interval])))`
- Objects synced: `sum by(type)(increase(pdbplus_sync_type_objects_total{type=~"$type"}[$__rate_interval]))`
- Deletes: `sum by(type)(increase(pdbplus_sync_type_deleted_total{type=~"$type"}[$__rate_interval]))`
- Errors: `sum by(type)(rate(pdbplus_sync_type_fetch_errors_total{type=~"$type"}[$__rate_interval]))`

Row 4 (Go Runtime) PromQL:
- Goroutines: `go_goroutine_count`
- Heap memory: `go_memory_used_bytes`
- Allocation rate: `rate(go_memory_allocated_bytes_total[$__rate_interval])`
- GC goal: `go_memory_gc_goal_bytes`

Row 5 (Business Metrics) PromQL:
- Object counts: `pdbplus_data_type_count{type=~"$type"}`
- Total objects: `sum(pdbplus_data_type_count)`
