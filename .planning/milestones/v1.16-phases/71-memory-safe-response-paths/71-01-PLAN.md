Current envelope shape (internal/pdbcompat/response.go):
```go
type envelope struct {
    Meta any `json:"meta"`
    Data any `json:"data"`
}
const poweredByHeader = "PeeringDB-Plus/1.1"
```

Current WriteResponse (the one we are replacing at Plan 04, not here):
```go
func WriteResponse(w http.ResponseWriter, data any) {
    w.Header().Set("Content-Type", "application/json")
    w.Header().Set("X-Powered-By", poweredByHeader)
    resp := envelope{Meta: struct{}{}, Data: data}
    _ = json.NewEncoder(w).Encode(resp)
}
```

CONTEXT.md D-01 contract verbatim:
1. Write `{"meta":` + `json.Marshal(meta)` + `,"data":[`
2. For each row from iterator: write `,` (if not first), then `json.Marshal(row)` to `w`
3. Write `]}` + flush
4. `http.Flusher.Flush()` called every N rows (default 100) for backpressure

rowsIter shape locked by orchestrator: `func() (row any, ok bool, err error)` — a pull-style closure. `ok=false, err=nil` signals clean EOF. `err != nil` aborts; the caller treats this as an unrecoverable stream error (nothing to do after bytes are on the wire — the response is already committed, so we return the error to the caller and let it be logged).

    Create `internal/pdbcompat/stream.go`. Use json.Marshal (not json.NewEncoder) per row because we want bytes without a trailing newline — the newline json.NewEncoder appends would corrupt our hand-rolled token stream.

    Do NOT wire `ctx` into a per-row Done() check yet; spec in CONTEXT.md D-01 does not ask for it and the iterator is driven by an ent.All() or future rows-chunk call that already respects ctx at its boundary. Document the deferred intent in the godoc.

    File skeleton:
    ```go
    package pdbcompat

    import (
        "context"
        "encoding/json"
        "fmt"
        "net/http"
    )

    // RowsIter is a pull-style row iterator. ok=false + err=nil signals clean EOF.
    // err != nil aborts the stream with an error returned to the caller.
    type RowsIter = func() (row any, ok bool, err error)

    // FlushEvery is the row count between periodic http.Flusher.Flush() calls.
    const FlushEvery = 100

    // StreamListResponse writes a PeeringDB envelope {"meta":...,"data":[...]}
    // token-by-token ... (full godoc mirroring CONTEXT.md D-01)
    func StreamListResponse(ctx context.Context, w http.ResponseWriter, meta any, rowsIter RowsIter) error {
        _ = ctx // reserved for future per-row cancellation; no-op today
        metaBytes, err := json.Marshal(meta)
        if err != nil {
            return fmt.Errorf("marshal meta: %w", err)
        }
        w.Header().Set("Content-Type", "application/json")
        w.Header().Set("X-Powered-By", poweredByHeader)
        if _, err := w.Write([]byte(`{"meta":`)); err != nil {
            return fmt.Errorf("write prelude: %w", err)
        }
        if _, err := w.Write(metaBytes); err != nil {
            return fmt.Errorf("write meta: %w", err)
        }
        if _, err := w.Write([]byte(`,"data":[`)); err != nil {
            return fmt.Errorf("write data-open: %w", err)
        }

        flusher, _ := w.(http.Flusher)
        idx := 0
        for {
            row, ok, iterErr := rowsIter()
            if iterErr != nil {
                return fmt.Errorf("stream row %d: %w", idx+1, iterErr)
            }
            if !ok {
                break
            }
            idx++
            if idx > 1 {
                if _, err := w.Write([]byte{','}); err != nil {
                    return fmt.Errorf("write row-sep %d: %w", idx, err)
                }
            }
            rowBytes, err := json.Marshal(row)
            if err != nil {
                return fmt.Errorf("marshal row %d: %w", idx, err)
            }
            if _, err := w.Write(rowBytes); err != nil {
                return fmt.Errorf("write row %d: %w", idx, err)
            }
            if flusher != nil && idx%FlushEvery == 0 {
                flusher.Flush()
            }
        }
        if _, err := w.Write([]byte("]}")); err != nil {
            return fmt.Errorf("write data-close: %w", err)
        }
        if flusher != nil {
            flusher.Flush()
        }
        return nil
    }
    ```

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-71-01-01 | I (Information disclosure) | json.Marshal(row) path — row types come from ent serializers | accept | Existing pdbcompat serializers already implement the Phase 64 privfield.Redact field-level privacy at construction time; the stream writer is just a transport and does not need its own privacy hook. Same trust boundary as current WriteResponse. |
| T-71-01-02 | D (Denial of service) | Iterator that never terminates (`ok=true, err=nil` forever) | mitigate | Plan 03 adds a pre-flight row-count budget check that caps the expected iteration length. An "always-true" iterator is not a reachable code path from any current caller. |
| T-71-01-03 | T (Tampering) | http.ResponseWriter Write error silently swallowed in a loop | mitigate | Every w.Write call checks err and returns wrapped. Prevents writing garbage bytes after a client disconnect by failing fast on first write error. |
