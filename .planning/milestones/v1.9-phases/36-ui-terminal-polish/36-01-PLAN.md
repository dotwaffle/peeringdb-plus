**Breadcrumb Component (per UI-05):**

Add a new `Breadcrumb` templ component to `detail_shared.templ`:

```go
// Breadcrumb renders navigation breadcrumbs: Home > TypePlural > EntityName.
// typePlural is the display name for the entity type (e.g., "Networks").
// typeURL is unused (no per-type list pages exist); the middle segment is non-linked text.
// entityName is the specific entity being viewed.
templ Breadcrumb(typePlural string, entityName string) {
    <nav aria-label="Breadcrumb" class="text-sm mb-4">
        <ol class="flex items-center gap-1.5 text-neutral-400">
            <li><a href="/ui/" class="hover:text-emerald-400 transition-colors">Home</a></li>
            <li class="text-neutral-500">&gt;</li>
            <li class="text-neutral-300">{ typePlural }</li>
            <li class="text-neutral-500">&gt;</li>
            <li class="text-neutral-100">{ entityName }</li>
        </ol>
    </nav>
}
```

Note: Using `text-neutral-500` for separator (passes WCAG AA at 4.7:1 on neutral-900). Middle segment is non-linked `<li>` since there are no per-type list pages. Using `text-neutral-300` for the type name (passes easily).
